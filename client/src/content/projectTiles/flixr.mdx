Flixr helps with managing tv show notifications for so many different streaming platforms.
It collates data from <u>online API's</u> to provide one succinct user interface for subscribing to TV show releases.
It uses a custom password-less authentication solution to accommodate <u>frictionless on-boarding</u>.
